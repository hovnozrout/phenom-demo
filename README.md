# Phenom demo
Three.js FPS game like tech demo, originally made for internal quarter presentations in Seznam.cz, a.s.

## Installation
1. Install all npm packages:
`npm i`

2. Install dev server:
`npm i -g webpack-dev-server`

3. Run dev server:
`webpack-dev-server`
